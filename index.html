Nemojs

selenium-webdriver automation for node.js

Introduction

Nemo will help create a flexible and maintainable automation solution for any node.js based project. While all the examples maintained by the team illustrate Nemo used with Mocha and GruntJS, Nemo can be grouped with ANY test-runner/task-runner software.

Getting Started

You can get started in a few ways.

  • Watch the screencasts below
  • clone the nemo-example-app kraken-js application, with a Nemo suite built in
  • Use generator-nemo yeoman generator to add Nemo to an existing kraken-js application
  • Dive into the details in the markdown documentation
